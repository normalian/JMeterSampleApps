# JMeter で ASP.NET MVC と ASP.NET WebForm の性能測定を試すプロジェクト
Java側ではデファクトスタンダード化している JMeter を利用して性能測定をするためのサンプルアプリ群になります。

* JMeterApp.WebForm … WebForm アプリ
* JMeterMvcApp … MVC アプリ
* PortableClasses … 共通のモデルクラス

## 前提環境
 * Visual Studio 2012
 * ASP.NET MVC4
 * JMeter 2.6以降

## 注意点
 * 簡略化のためにデータベース等を利用していません
